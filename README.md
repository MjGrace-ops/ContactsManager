# ContactsManager

Projet Java permettant :
- d'ajouter des contacts,
- de rechercher un contact par nom.

## Fonctionnalités
- Ajout de contacts
- Recherche par nom
- Affichage du numéro de téléphone

## Technologies
- Java

## Exécution
1. Compiler les fichiers :
```bash
javac *.java
```

2. Exécuter :
```bash
java Main
```
